# Welcome to MultiTenantKit Documentation

MultiTenantKit is a production-ready TypeScript toolkit that provides the **business logic** for managing **Users**, **Organizations**, and **Organization Memberships** in B2B SaaS applications.

## Why MultiTenantKit?

- ✅ **18 production-ready use cases** out of the box
- ✅ **Your data stays in your infrastructure**
- ✅ **Type-safe custom fields** with Zod schemas
- ✅ **Hexagonal architecture** for maximum flexibility
- ✅ **Zero lock-in** - it's your code
