## Make It Yours

Need to store additional data? Add custom fields with type-safe Zod schemas:

```typescript
import { createSupabaseExpressApp } from '@multitenantkit/sdk';
import { z } from 'zod';

const app = createSupabaseExpressApp({
  users: {
    customFields: {
      customSchema: z.object({
        firstName: z.string(),
        lastName: z.string(),
        avatar: z.string().url().optional(),
      }),
    },
  },
  organizations: {
    customFields: {
      customSchema: z.object({
        industry: z.string().optional(),
        size: z.enum(['startup', 'small', 'medium', 'enterprise']).optional(),
      }),
    },
  },
});

app.listen(3000);
```

Want to send a welcome email when a user is created? Use hooks:

```typescript
const app = createSupabaseExpressApp({
  useCaseHooks: {
    CreateUser: {
      onSuccess: async ({ stepResults }) => {
        await sendWelcomeEmail(stepResults.output.email);
      },
    },
  },
});
```
